# Devedu-labedit

Prerequisites:
Install (node)[https://nodejs.org/en/].

Clone the repository.

Install dependencies by running:
```
$ npm install
```

Add a `.env` file to the root of the repository. Ask Gabe Matute for access.

Build webpack:

```
$ npm run build
```

Then start with:
```
$ npm start
```

The server should be live at localhost:3000.
